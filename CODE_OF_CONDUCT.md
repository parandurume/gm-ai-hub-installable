# Code of Conduct / 행동 강령

## Our Pledge / 우리의 약속

We are committed to making participation in this project a welcoming and
respectful experience for everyone. This project exists to help public officers
serve their communities better — and our community should reflect that spirit
of public service.

## Standards / 기준

**Expected behavior:**
- Be respectful, constructive, and patient
- Welcome newcomers, especially public officers trying open source for the first time
- Give and accept feedback gracefully
- Focus on what is best for the community and for public service

**Unacceptable behavior:**
- Harassment, discrimination, or personal attacks
- Publishing others' private information
- Disruptive or deliberately unconstructive behavior
- Using the project to spread misinformation about any government or institution

## Scope / 범위

This Code of Conduct applies to all project spaces: GitHub issues, pull requests,
discussions, and any related communication channels.

## Enforcement / 시행

Instances of unacceptable behavior may be reported to the project maintainers.
All reports will be reviewed and handled appropriately.
